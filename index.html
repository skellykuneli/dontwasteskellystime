Don’t Waste Skelly’s Time

Be cool. Be direct. Say what you need.

Just ask the question.

You don’t need to ask if you can ask. You don’t need to check if I’m around. Put the question in the first message so I can actually help.

Don’t send this

yo Hello, are you around? hi skelly- quick question. You got a sec?

Send this instead

yo skelly I have an issue with "x" or yo skelly could you tell me "x"

Even better

yo skelly I have an issue with "x" I tried "y" I expected "z" but this happened instead: "error/message/result"

The rules

  • Say what you need in the first message.
  • Include context if it matters.
  • Show what you already tried.
  • Don’t make me play message tag.
